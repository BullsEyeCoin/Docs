Telegram is where the action gets posted fast.

## What the bot does

- Posts bonding pot reminders before migration
- Posts qualifying buy results after migration
- Highlights winners
- Posts jackpot reminder messages once the game is live
- Links users to on-chain verification

## Why it matters

The bot is not the truth machine. The blockchain is.

The bot exists so the group can instantly see:

- whether the vault is still bonding or already live
- who took a shot
- whether it hit or missed
- how big the jackpot is now
- where to verify it on-chain

## Reminder messages

Reminder posts can be used to:

- show the bonding pot before migration
- keep attention on the live jackpot after migration
- point users to buy and chart links
- reinforce the public game loop around the vault

## Media

The bot can run text-only or media-backed. Either way, the chain should always back up the post.
