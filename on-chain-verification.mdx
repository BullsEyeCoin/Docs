# On-Chain Verification

Bulls Eye is built so the core mechanics can be checked on-chain.

## What users can verify

- Taxed buys and sells
- Vault funding activity
- Qualifying jackpot attempts
- Winner payouts
- Live jackpot state

## What the Telegram feed represents

The Telegram bot is meant to mirror the on-chain result flow for qualifying buys.

That means:

- result posts are tied to on-chain activity
- winner messages correspond to on-chain resolution
- verification links can point directly to the relevant transaction

## Why this matters

The project works best when the community can independently confirm:

- whether a buy qualified
- whether it won or lost
- whether the payout actually happened
- how the vault changed over time

The goal is to keep the jackpot system transparent enough that the public game can be followed without relying only on social posts.
